## 개발 환경
AWS ec2

### 개발환경 셋팅
```bash
# git clone
$ git clone https://github.com/krafton-codeCraft/code-craft.git
# install npm
$ brew install npm
# java
brew install openjdk@17
```
### 실행 방법
```bash
chmod +x startlocal.sh
./startlocal.sh
```
